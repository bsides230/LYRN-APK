Recommended startup order (important)

  • Start the model first using normal LYRN controls in the main app.
  • Then open the Claude module.
  • Set workspace/flags, then press Reconnect if needed to attach terminal.
  • Finally launch Claude Code with the purple launch button.

Mobile behavior

On phones/tablets, the left panel becomes a slide-in drawer. Use the top-left ☰ button to open it, then scroll the panel for all controls while terminal remains full-width behind it.
